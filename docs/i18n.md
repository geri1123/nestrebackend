## LanguageMiddleware

Purpose:
- Detect user language for each HTTP request
- Store it in `req.language` and `requestContext`

Language resolution priority:
1. `x-lang` header
2. `accept-language` header
3. `NEXT_LOCALE` cookie
4. `?lang=` query param
